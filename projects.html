AURA — Multimodal Emotion Recognition System

RoBERTa Wav2Vec2 GPT-4o FastAPI

What

End-to-end multimodal pipeline that fine-tunes transformer models on text and speech to detect emotion, then dynamically shapes an LLM's response tone via FastAPI.

Impact

  • 7-class emotion classification (text + speech)
  • α-weighted late fusion of multimodal vectors
  • Sub-second inference latency via FastAPI
  • Pipeline: Whisper ASR → ERC → fusion → policy → LLM

Stack

Python, PyTorch, HuggingFace Transformers, RoBERTa (GoEmotions), Wav2Vec2 (CREMA-D), OpenAI API, FastAPI, Whisper ASR, librosa, Docker

SHEO — Smart Home Energy Optimizer

Pyomo GLPK Linear Programming Optimization

What

Multi-objective LP optimizer that schedules household appliance usage across 24 hourly intervals under a 7 kW power cap — balancing cost, carbon, and comfort with configurable user preferences.

Impact

  • 3 user profiles: cost_saver, eco_warrior, comfort_first
  • 15–30% projected electricity cost savings
  • Contract-aware tariff logic (per-state pricing)
  • Structured JSON output for downstream integration

Stack

Python, Pyomo, GLPK solver, JSON schema design, optimization modeling
